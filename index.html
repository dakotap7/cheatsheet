Color Formats

Style Format Details Usage
Named Colors color: aqua; English words that describe colors, also called keyword colors This is most usefal for testing and demonstration purposes, but overall it's a limited palette.
Hexidecimal color: #E9D758; A 6-digit hex code prefixed with # contains three 2-digit values, one for each channel (red / green / blue) Hexidecimal values is the most common format that you'll see used.
RGB color: rgb(255 158 177); Numeric values that describe a mix of red, green, and blue In web design we're primarily concerned with what web pages look like on screens, RGB is the color model we use.
HSL color: hsl(303 87% 73%) Numeric values that describe a mix of hue, saturation, and lightness The HSL color model is one of the least used, but gaining traction because can be more intuitive to use when working with shades and color adjustments.

Format Variations

Style Variation Format Details
Hexidecimal Shorthand color: #000; If both digits of the red, green and blue values are the same, you may use the short three-digit notation.
RGBA color: rgba(255 158 177 0.5); It's just like RGB, except with the addition of a fourth value: the alpha channel. The alpha value represents the level of transparency that the rgb color should have. It can be a value from 0 to 1 or a percentage from 0 to 100%.
HSLA color: hsla(303 87% 73% 0.75); HSLA is simply the HSL color model with the addition of an alpha channel. This works exactly the same way as the alpha channel in RGBA.
